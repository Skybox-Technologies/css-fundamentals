Layouting exercises

Grid

Let's arrange these boxes neatly in an evenly spread 5 by 3 grid!

Solution: Apply display: grid; grid-template-columns: repeat(5, 1fr); grid-template-rows: repeat(3, 1fr); gap: 10px; to the container to define the grid.

Let's mix it up! Col 1 should be 1fr, col 2 should be a fixed 100px, col 3 should be 30%, col 4 should be 10em and let col 5 be 2fr! Meanwhile, let's make the first row 5rem tall, the second row 250px tall and the third row 1fr. Let's put a gap of 10px and have the boxes stretch to fill out each cell!

(You might have to unset the width and height on the boxes themselves to achieve the requested result)
Solution: Apply display: grid; grid-template-columns: 1fr 100px 30% 10em 2fr; grid-template-rows: 5rem 250px 1fr; gap: 10px; to the container to define the grid.

Time to try an area map! Given a 5 by 5 grid (you may define the template as you like), define an area template that reserves the top row for a header, the bottom row for a footer, the 3 left-most cells in the middle rows as the sidebar and the rest as the main content areas. Assign our four boxes to their designated areas.

(You might have to unset the width and height on the boxes themselves to achieve the requested result)
Solution: Apply display: grid;, define the template columns and rows, then add a grid-template-areas definition and assign the grid-areas to the cells

Let's quickly explore how grid can be used to center an element, too!

Solution: As with flex, we can use justify-content: center and align-items: center to achieve perfect centering with grid!

Just for funzies, let's make a vertical pyramid by layering boxes on top of each other!

(Works best if you re-define widths and heights for each box to be ever-decreasing; you pick a size!)
Solution: Apply display: grid; place-items: center; and a template of just 1 row and 1 columns to the container, then specify grid-column: 1 and grid-row: 1 as the desired column and row for each box to layer the boxes on top of each other.
