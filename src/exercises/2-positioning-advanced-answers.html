Advanced positioning exercises

The position property

We want the colored box to be roughly in the center while keeping it in the regular document flow. Can you fix it?

(Reminder: position relative unlocks offset controls)
Solution: Add position: relative; top: 35%; left: 45%; to position the box.

The boss says it's okay to remove the box from the normal document flow. He still wants it centered. Ideas?

Solution: Add the same top and left styles as before, but apply a position: absolute to the box. Remember to also apply a position: relative to its parent!

Okay, nevermind! The box should actually be a sticky header! It should take up the whole width of the browser window and only have a height of 30px. and stick to the top of the page... can you fix it?

Solution: Apply position: fixed to the box to take it out of normal flow and position it in relation to the viewport. Add top: 0; left: 0; right: 0; to position it at the top of the viewport. Resize it with width: auto; height: 30px;

So... this box needs to be within the boundaries of the container, but it needs to stick to the top of the page, 20px below the header, when scrolling until the container scrolls out of view, at which point it shoud follow the container out of view.

Solution: Apply position: sticky to the box and give it a top offset of top: 50px to make it start to stick 20px below the 30px tall header.

Alternatives to position

Here, we want to center the box, but due to external constraints, we don't want to use relative positioning. Try leveraging margin instead.

(Percentage-wise margin values are in relation to the container - not the element itself)
Solution: Apply margin: 20% auto to the box

In this case, we have a few boxes in a row. Try shifting the middle one towards the right and the bottom by half of its size in both directions using transform: translate.

(Remember: transform doesn't affect the document flow and works in relation to the element itself, not its parent)
Solution: Apply transform: translateX(50%) translateY(50%) to the box

The best and most precise way of centering content nowadays is with the combination of offset controls in conjunction with translate: transform. Can you center the box using these methods?

Solution: Apply position: relative; top: 50%; left: 50%; transform: translateX(-50%) translateY(-50%) to the box
