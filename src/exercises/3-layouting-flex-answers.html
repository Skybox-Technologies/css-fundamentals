Layouting exercises

Flex

We want the colored boxes arranged in a row with equal spacing between them, centered vertically. Can you do it?

Solution: Add display: flex; justify-content: space-between; align-items: center; to the container to arrange the boxes in a row with equal spacing between them and center them vertically.

Using flex, center the box.

Solution: Add display: flex; justify-content: center; align-items: center; to the container to center the box.

For this next exercise, align all the boxes along the right-hand and left-hand sides of the container, alternating between sides. Leave a gap of 20px between the boxes.

Solution: Add display: flex; flex-direction: column; align-items: flex-end; gap: 20px; to the container to align the boxes along the right-hand side of the container. Use align-self: flex-start; on every other box to align them along the left-hand side.

There are too many boxes for the container, which is crushing them together! How can we fix this, so the boxes keep their intended size?

Solution: Add flex-wrap: wrap; to the container to break the boxes down onto new lines.
